API Movies

Endpoints

Movies API

Request URL Response
GET /movies List of all movies
GET /movies/:title Details of a specific movie
GET /movies/:genre/genreName Details of a specific movie genre
GET /movies/directors/:directorName Director of a specific movie
POST /users/:id/:movieTitle Confirmation of movie added to Favorites
DELETE /users/:id/:movieTitle Confirmation of movie deletion

Users API

Request URL Response
POST /users Confirmation of user creation
PUT /users/:id Confirmation of user update
DELETE /users/:id Confirmation of user deletion
